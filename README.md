# Password Generator

## Overview

A simple Password Generator application built using Python and Tkinter. The application generates a password based on the user's name by combining it with random numbers and special characters.

## Features

* User-friendly graphical interface
* Accepts user name as input
* Generates unique passwords
* Uses random numbers and special characters
* Error handling for empty input

## Technologies Used

* Python
* Tkinter

## Run the Application

```bash
python password_generator.py
```
